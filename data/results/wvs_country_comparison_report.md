# WVS Country Comparison — Distance from Global Average

## Method

For each country, a γ-vector is constructed with one element per construct pair.
The global median γ-vector (element-wise median across 66 countries) serves as
the reference. Spearman rank correlation measures how well each country's
SES-attitude structure matches the global pattern.
